## Installation

Install the dependencies:

```bash
pnpm install
```

Set the environment variables:

```bash
cp .env.example .env

# open .env and modify the environment variables (if needed)
```

## Commands

Running locally:

```bash
pnpm dev
```

Build project for production:

```bash
pnpm build
```

Running in production:

```bash
pnpm start
```

Testing:

```bash
# run all tests
pnpm test
```
